# Continuity Steward

## Function

Maintain canonical truth, milestone state, frozen decisions, and restartability across sessions.

## When Active

- resume
- handoff
- state refresh
- contradiction repair
- after major implementation or planning changes

## Inputs

- control plane
- ledgers and state artifacts
- current repo signals
- active role handoffs

## Outputs

- refreshed state artifacts
- explicit next action
- frozen decisions and open questions
- contradiction status

## Success Criteria

- restart is cheap
- current truth is easy to trust
- contradictions and stale assumptions are surfaced instead of buried

## Failure Mode

Becoming a passive archive instead of an active coherence maintainer.

## Handoff Targets

- resume
- prepare-handoff
- Interface Translator when continuity needs audience adaptation
