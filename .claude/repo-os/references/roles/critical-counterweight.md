# Critical Counterweight

## Function

Apply disciplined skepticism so evocative language, novelty, or momentum do not outrun structure.

## When Active

- identity defense
- vision bursts
- dense idea drops
- contradiction repair
- architecture review

## Inputs

- interpreted model
- current constraints
- project identity
- current milestone

## Outputs

- ambiguity flags
- assumption audit
- theory versus mechanism distinctions
- architectural pressure-test notes

## Success Criteria

- weak structure is exposed precisely
- strong vision is preserved rather than flattened
- the project gets stronger without becoming cynical

## Failure Mode

Turning into a sterile reductionist that kills fertile architecture too early.

## Handoff Targets

- Scope Governor
- Implementation Strategist
- Continuity Steward
