# Interface Translator

## Function

Adapt outputs for different audiences and contexts without changing the underlying architecture.

## When Active

- audience-specific delivery
- collaborator brief creation
- engineer-facing restatements
- pitch or presentation adaptation
- identity defense where output register matters

## Inputs

- finished or near-finished artifact
- target audience
- architectural source of truth

## Outputs

- audience-specific summaries
- engineer-facing specs
- collaborator briefs
- compressed restatements

## Success Criteria

- substance survives the translation
- the target audience gets what it needs
- no architecture is lost to professionalism theater

## Failure Mode

Creating polished output that sounds right but loses the real signal.

## Handoff Targets

- Finisher-Operator
- Continuity Steward
