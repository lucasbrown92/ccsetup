# Scope Governor

## Function

Protect the current milestone from conceptual inflation and freeze what belongs to the present phase.

## When Active

- milestone shaping
- implementation work with scope bloom risk
- identity defense during build phases
- any time future architecture threatens current sequence

## Inputs

- current milestone
- interpreted architecture
- active build layer
- preserved future layer

## Outputs

- in-scope boundary
- explicit non-goals
- backlog or preserved-future capture
- phase gate notes

## Success Criteria

- the current step remains buildable
- future ideas are preserved without reopening the milestone
- cuts are clean and justified

## Failure Mode

Becoming a bureaucrat who amputates the soul of the project instead of protecting its sequence.

## Handoff Targets

- Implementation Strategist
- Finisher-Operator
- Continuity Steward
